Phidget Capacitive Touch.lvlib:CapacitiveTouchStart.vi

Opens and initializes a Phidget Capacitive Touch

 Change Trigger

The change trigger value

1015_0 (ver: 500, 101, 102) Capacitive Slider Sensor: Min: 0 Max: 1 Def: 0.005

1016_0 (ver: 510, 101, 102) Capacitive Scroll Sensor: Min: 0 Max: 1 Def: 0.005

HIN1000_0 (ver: 104) Capacitive Touch Sensor: Min: 0 Max: 1 Def: 0

HIN1001_0 (ver: 103) Capacitive Touch Sensor: Min: 0 Max: 0.5 Def: 0

HIN1001_0 (ver: 103) Capacitive Scroll Wheel: Min: 0 Max: 0.5 Def: 0

Data Interval

The data interval value

Units: milliseconds (ms)

1015_0 (ver: 500, 101, 102) Capacitive Slider Sensor: Min: 60 Max: 1000 Def: 60

1016_0 (ver: 510, 101, 102) Capacitive Scroll Sensor: Min: 60 Max: 1000 Def: 60

HIN1000_0 (ver: 104) Capacitive Touch Sensor: Min: 25 Max: 1000 Def: 25

HIN1001_0 (ver: 103) Capacitive Touch Sensor: Min: 20 Max: 250 Def: 20

HIN1001_0 (ver: 103) Capacitive Scroll Wheel: Min: 20 Max: 250 Def: 20

error in (no error)

error in can accept error information wired from VIs previously called.

    		status

    status is TRUE (X) if an error occurred or FALSE (checkmark) to indicate a warning or that no error occurred.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    		code

    code is the error or warning code.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

    		source

    source describes the origin of the error or warning.

    Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

Device Info

Information required to specify which Phidget to open

    		Serial Number

    The device serial number

    		Channel

    The channel index

    		Port

    The hub port index

Connection

Selects whether to look for a Phidget on the Local machine, over a Remote network, or both (No Preference)

Timeout (mSec)

Timeout in milliseconds

Attached?

Returns TRUE is the device successfully attached, or FALSE otherwise

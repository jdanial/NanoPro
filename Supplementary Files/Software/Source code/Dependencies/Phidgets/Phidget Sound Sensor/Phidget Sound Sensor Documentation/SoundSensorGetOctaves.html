Phidget Sound Sensor.lvlib:SoundSensorGetOctaves.vi

The unweighted value of each frequency band.

  • The following frequency bands are represented:
    • octaves[0] = 31.5 Hz
    • octaves[1] = 63 Hz
    • octaves[2] = 125 Hz
    • octaves[3] = 250 Hz
    • octaves[4] = 500 Hz
    • octaves[5] = 1 kHz
    • octaves[6] = 2 kHz
    • octaves[7] = 4 kHz
    • octaves[8] = 8 kHz
    • octaves[9] = 16 kHz

Magnetometer Instrument Driver Readme

1. Overview

Instrument Driver Technology: LabVIEW Plug and Play (project-style)
Manufacturer: Phidgets Inc.
Supported Language(s): LabVIEW
Supported Model(s): 1042_0, 1044_0, 1056_0, 1056_0, MOT1101_0
Model(s) Tested: 1042_0, 1044_0, 1056_0, 1056_0, MOT1101_0
Interface(s): USB, VINT

Certified: No
NI Supported: No
Source Code Available: No

Driver Revision: 1.0.0.20190730
Original Release Date: 06/26/2017
Current Revision Date: 07/30/2019

2. Required Software

Some software components need to be installed before using this instrument driver. The minimum versions of these components are listed below, and can be downloaded from the Download Site.

Phidget22 Core Libraries, Bonjour Print Services (if network connectivity is required)

Refer to the LabVIEW Help for more information about software requirements. You access the LabVIEW Help by selecting Help»Search the LabVIEW Help.

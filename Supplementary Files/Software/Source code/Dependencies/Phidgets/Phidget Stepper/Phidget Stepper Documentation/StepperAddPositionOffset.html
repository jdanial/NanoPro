Phidget Stepper.lvlib:StepperAddPositionOffset.vi

Adds an offset (positive or negative) to the current position and target position.

  • This is especially useful for zeroing position.

    • Device Support:

    1062_0 (ver: 101) Unipolar Stepper Controller : Supported

    1062_1 (ver: 101, 102) Unipolar Stepper Controller : Supported

    1063_0 (ver: 100) Bipolar Stepper Controller : Supported

    1063_1 (ver: 100, 101) Bipolar Stepper Controller : Supported

    1067_0 (ver: 200, 201) Bipolar Stepper Controller : Supported

    STC1000_0 (ver: 100) Bipolar Stepper Controller : Supported

    STC1001_0 (ver: 100) Bipolar Stepper Controller : Supported

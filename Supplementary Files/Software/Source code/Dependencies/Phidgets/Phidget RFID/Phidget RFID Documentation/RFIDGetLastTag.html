Phidget RFID.lvlib:RFIDGetLastTag.vi

Gets the most recently read tag's data, even if that tag is no longer within read range.

  • Only valid after at least one tag has been read.

    • Device Support:

    1023_0 (ver: 103) RFID Reader : Supported

    1023_0 (ver: 104) RFID Reader : Supported

    1023_0 (ver: 200) RFID Reader : Supported

    1023_0 (ver: 201, 202, 203, 204) RFID Reader : Supported

    1023_1 (ver: 205, 206) RFID Reader : Supported

    1024_0 (ver: 100, 101, 103) RFID Reader/Writer : Supported

Phidget RFID.lvlib:RFIDWrite.vi

Writes data to the tag being currently read by the reader.

  • You cannot write to a read-only or locked tag.

    • Device Support:

    1023_0 (ver: 103) RFID Reader : Unsupported

    1023_0 (ver: 104) RFID Reader : Unsupported

    1023_0 (ver: 200) RFID Reader : Unsupported

    1023_0 (ver: 201, 202, 203, 204) RFID Reader : Unsupported

    1023_1 (ver: 205, 206) RFID Reader : Unsupported

    1024_0 (ver: 100, 101, 103) RFID Reader/Writer : Supported

    		 Device In

    		error in (no error)

    error in can accept error information wired from VIs previously called.

      		status

      status is TRUE (X) if an error occurred or FALSE (checkmark) to indicate a warning or that no error occurred.

      Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

      		code

      code is the error or warning code.

      Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.

      		source

      source describes the origin of the error or warning.

      Right-click the error in control on the front panel and select Explain Error or Explain Warning from the shortcut menu for more information about the error.
